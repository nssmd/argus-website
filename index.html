系统 · 三角色 agent

一个笨管道，三个 agent，胜负永远外置。

管道不替 agent 思考——它搬运状态、判定形式、把胜负留给外部裁决。Planner 定方向，Engineer 写代码和跑实验，Reviewer 当对手审。三个角色各跑各的 LLM，谁也不替谁背书。

聪明的部分在 agent，诚实的部分在管道。
