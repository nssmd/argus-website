SOL-ExecBench（B200）：argus 冲到全球第六，两个 kernel 拿下全榜第一

SOL-ExecBench 是 NVIDIA 的公开 GPU 算子基准（235 个真实 kernel · B200）。SOL-Score 根据 SOLAR 分析得到的 B200 理论 roofline 评价自定义 kernel，越高越好。argus 提交了其中 101 个 kernel，全部由 NVIDIA 在 B200 上实测；这轮 campaign 的综合排名达到全球第六，7 个 kernel 曾进入公开榜前三，其中 2 个全榜第一。
✅ 在 NVIDIA 官方 SOL-ExecBench B200 榜上，argus 的综合表现达到全球第六。其中 040_conv2d_residual_block（0.851） 与 002_fp8_attention_qkv_projection（0.708） 是全榜第一，分别压过 SF Tensor / Databricks（0.846）与 Recursive（0.647）。所有分数均由 NVIDIA 在 B200 上实测，任何人都可以在公开榜复核。完整战绩：101 个提交、7 个 kernel 曾进公开前三、2 个全榜第一。榜单持续变化；“全球第六”记录的是这轮 campaign 达到的综合名次。

分数怎么来的（为什么这次更硬）

• SOL-Score 以 SOLAR 分析得到的 B200 理论 roofline 为参照评价 custom kernel；越高越好。

• 流程：提交 kernel → NVIDIA 在 B200 上跑并计分 → 上公开榜。分数不是我们自己机器上测的，而是第三方（NVIDIA）在标准 B200 上测的。

• 因此任何人都能在公开榜上独立复核我们的名次——这比我们自建 harness 自测的数更可信，正是「自报 → 外部可核」的那一步。

榜单口径

• 提交了 101 / 235 个 kernel，NVIDIA 全部实测，这轮 campaign 的综合名次达到全球第六，7 个 kernel 曾进公开前三。我们没有刷满 L1 / L2 / Quantization / FlashInfer 任一 collection，因此“全球第六”不等于 collection 冠军。

• 「全榜第一」= 该 kernel 目前已知最好提交（榜每 kernel 取最优）；多数核我们是 #2/#3，#1 仍是对手。

• 与 Recursive 头对头，两边都照实说：fp8_attention 我们赢（0.708 vs 0.647）、flux_rmsnorm 我们险胜（0.5801 vs 0.5796）；不过榜是动态的，别人也在交、也在超我们。
与其它实验的关系：nanochat 考「能不能发明新机制」、speedrun 考「能不能在近-SOTA 配方上继续抠时间」、这里 SOL-ExecBench 考「能不能把真实算子逼近硬件光速」——而且是唯一一个分数由外部第三方（NVIDIA）在标准硬件上实测的 vertical。同一套自治研究引擎，三种难度形态。
